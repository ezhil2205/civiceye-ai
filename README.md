# CivicEye AI – Smart Civic Issue Reporting Platform

## Overview

CivicEye AI is an AI-powered Smart City web application designed to bridge the gap between citizens and municipal authorities. The platform enables users to report civic issues such as garbage accumulation, broken streetlights, potholes, and water leakages using live camera capture and GPS location tracking.

By leveraging modern web technologies and AI-assisted features, CivicEye AI aims to create cleaner, safer, and smarter cities through citizen participation.

---

## Problem Statement

Citizens often encounter civic issues in their neighborhoods but lack a simple and efficient way to report them to local authorities. Delayed reporting can lead to environmental, safety, and public health concerns.

CivicEye AI addresses this challenge by providing a centralized platform for reporting and tracking civic issues in real time.

---

## Key Features

### 📸 Live Camera Capture

* Capture images directly using the device camera.
* No gallery uploads allowed to ensure authenticity.

### 📍 Automatic GPS Detection

* Fetches the user's current location using the Geolocation API.
* Displays location details for accurate issue reporting.

### 🤖 AI-Assisted Classification

* Classifies reported issues such as garbage, potholes, and streetlight failures.
* Assigns priority levels for efficient handling.

### 📊 Interactive Dashboard

* Displays complaint statistics and analytics.
* Tracks complaint status and progress.

### 🗺️ Map Integration

* Visualizes reported issues on an interactive map.
